# Requirements — PortfolioAI

最終更新: 2026-06-20

---

## 背景・目的

個人の資産運用データ（Excel）を手軽にビジュアライズし、資産推移・配分を一目で把握できるようにする。
現状は Excel だけで管理しており、グラフ確認が手間。Web アプリ化することで、スマホからも確認できるようにする。

---

## ユーザー

- 個人投資家 1 名（本人のみ利用）
- デバイス: MacBook / iPhone
- 技術レベル: 非エンジニア（Excel 操作は得意）

---

## 機能要件

### F-01 Excel アップロード（最優先）

- [ ] `.xlsx` / `.xls` ファイルをドラッグ&ドロップまたはファイル選択でアップロードできる
- [ ] アップロード前にファイル名・サイズ・シート数をプレビュー表示する
- [ ] 取込対象シートを ON/OFF で選択できる（デフォルトは自動認識）
- [ ] 取込モードを選択できる（追記 / 上書き）
- [ ] 基準日・タグを手動入力できる
- [ ] アップロード中は進捗バーを表示する
- [ ] 完了後はトースト通知を表示し、ダッシュボードへ遷移する
- [ ] 取込履歴（過去のアップロード一覧）を表示する
- [ ] エラー時はわかりやすいメッセージを表示する（日本語）

### F-02 ダッシュボード

- [ ] 総資産 / 含み益 / 損益率 / 前月比 を KPI カードで表示する
- [ ] 総資産の月次推移をラインチャートで表示する（期間: 全期間 / 3年 / 1年 の切替）
- [ ] 資産クラス別配分をドーナツチャートで表示する（最新スナップショット）
- [ ] カテゴリ別残高の推移を積み上げ棒グラフで表示する
- [ ] 資産配分割合をプログレスバーで表示する
- [ ] 保有銘柄一覧をテーブルで表示する（評価額・含み益・損益率）

### F-03 レポート出力（Phase 4）

- [ ] ダッシュボードを PDF でダウンロードできる
- [ ] 月次サマリーをメール送信できる（任意）

---

## 非機能要件

### パフォーマンス
- ダッシュボード初期表示: 2 秒以内（LCP）
- Excel 解析: 50MB ファイルを 10 秒以内に処理

### セキュリティ
- 個人利用のため認証は Phase 1 では不要
- ファイルは Vercel Blob に保存。URL は署名付き（外部公開しない）
- アップロードの MIME タイプ・拡張子チェックを行う

### 可用性
- Vercel Hobby プランで運用（SLA なし）

## データモデル（概要）

### `uploads` テーブル

| カラム | 型 | 説明 |
|---|---|---|
| id | uuid | PK |
| filename | text | 元ファイル名 |
| blob_url | text | Vercel Blob URL |
| base_date | date | 基準日 |
| tag | text | 任意タグ |
| sheets_processed | text[] | 取込したシート名 |
| records_inserted | int | 挿入レコード数 |
| created_at | timestamp | アップロード日時 |

### `monthly_trends` テーブル

| カラム | 型 | 説明 |
|---|---|---|
| id | uuid | PK |
| date | date | 月末日 |
| total | bigint | 総資産（円） |
| cash | bigint | 現金・預金・暗号資産 |
| stock | bigint | 株式（現物） |
| trust | bigint | 投資信託 |
| pension | bigint | 年金 |
| points | int | ポイント |
| upload_id | uuid | FK → uploads |

### `holdings` テーブル

| カラム | 型 | 説明 |
|---|---|---|
| id | uuid | PK |
| ticker | text | 分類コード（VTI / VWO / 金 等） |
| name | text | 銘柄名 |
| institution | text | 保有金融機関 |
| quantity | numeric | 保有口数 |
| avg_cost | numeric | 平均取得単価 |
| price | numeric | 基準価額 |
| value | bigint | 評価額（円） |
| profit | bigint | 評価損益（円） |
| profit_rate | numeric | 損益率 |
| upload_id | uuid | FK → uploads |

## 受け入れ条件

- `資産運用(080630).xlsx`（8シート、481KB）をアップロードして正常に取込できること
- ダッシュボードに総資産 ¥106,755,195、含み益 +¥40,266,694（+72.4%）が表示されること
- iPhone Safari で表示が崩れないこと（レスポンシブ対応）

## スコープ外（明示的に除外）

- 複数ユーザー対応
- マネーフォワード API 連携
- 取引履歴の入力フォーム
